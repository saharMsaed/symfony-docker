# Chiron

## Lancer le projet 

####  Requis sur la machine de développement :
* Docker et docker-compose
* Git

#### Démarrage quotidien

```bash
make start
```

#### Arrêt quotidien 

```bash
make stop
```

Les tests unitaires (TestCase ou KernelTestCase), fonctionnels (WebTestCase) :

```bash
make tests
```
